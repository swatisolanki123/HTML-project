# 📝 Registration Form

A simple and responsive **Registration Form** created using HTML and CSS.  
This project is designed to collect user information in a clean and structured format.

---

## 📌 Features

- User-friendly form design
- Input fields for user details
- Table-based layout (structured format)
- Responsive and clean UI
- Basic form validation using HTML

---


## 🛠️ Technologies Used

- HTML5
- CSS3
